Meet all 28 speakers at BSides Kraków 2026

18 July 2026 · 12 min read · By the BSides Kraków Team

The 2026 speaker roster connects deep technical research with the human, organisational and operational realities of security. Here is a concise introduction to every published speaker.

Use this guide to find the perspectives most relevant to your work, then review the latest schedule for session times and programme updates.

Cássio Pereira

BSides Kraków Founder

An application security expert and the founder and host of BSides Kraków, connecting the programme with the wider local security community.

Victoria Shutenko

Security Engineer and Penetration Tester at TechMagic

A cloud and application security practitioner with experience testing web and mobile applications, networks and AWS environments.

Kim Dvash

Offensive Security Team Leader, Israel Aerospace Industries

Mateusz Olejarka

Principal Security Consultant, SecuRing

A web application security specialist, experienced trainer and bug-bounty researcher with more than a decade in IT security.

David Papp

CEO, Gen0Sec

A cybersecurity entrepreneur working on AI-powered detection and response as well as protection for AI and LLM applications.

Michael Reimsbach

Product Security at SAP

Works with SAP Cloud Infrastructure security on vulnerability management, secrets management and secure internal services.

Rishi Rxerium

Security Researcher

Souvik Kandar, aka HackerMight

Saurabh Mishra

Lead Consultant

A programme built from different perspectives

The roster spans offensive and defensive security, application and cloud security, research, detection engineering, governance, AI, connected systems and community education. That range is deliberate: useful security decisions rarely belong to only one discipline.
